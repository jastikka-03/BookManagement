# BookManagement

## Description
BookManagement is a full-stack web application that allows users to **manage a list of books**. Users can add, view, update, and delete books. The application uses **Angular** for the frontend and **ASP.NET Core with C#** for the backend. The backend stores data in a simple in-memory list.

# Features
- View list of books
- Add new books
- Edit existing books
- Delete books

# Technologies
- *Frontend:** Angular, Bootstrap, HTML, CSS, TypeScript
- *Backend:** ASP.NET Core, C#
- *Data Storage:** In-memory list (temporary, no database)

## Getting Started

### Backend
1. Navigate to the backend folder.
2. Run the application

Frontend

Navigate to the frontend folder.

Install dependencies:

npm install

Run the Angular application:

ng serve

Open your browser
